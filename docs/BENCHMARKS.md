# Benchmarks

> Results are populated by `make bench` + `make bench-summary`. Do not edit this file manually.

## How to reproduce

```bash
# 1. Start 4-shard cluster
make compose-up

# 2. Index 1M documents
make index-1m

# 3. Run load test
make bench

# 4. Summarize results
make bench-summary
```
